PARA USAR EL TRIGGER PARA SPAWNEAR WAVES : EL BOOL DEL SECTIONNODE(START AT TRIGGER) DEBE ESTAR ACTIVADO.
EL TRIGGER(ENEMYSPAWNERTRIGGER) DE LA SECCION DEBE TENER CHEQUEADO EL BOOL TAMBIEN, Y LOS SPAWNERS NO DEBEN
TENER CHEQUEADO EL TRIGGER SPAWN (SI SE USA PARA SPAWNEAR WAVES NO HAY QUE PONER NADA EN EL GAMEOBJECT CONTAINER, MIENTRAS ESTE ACTIVADO EL BOOL DEL TRIGGER, SINO TIRARA ERROR)

PARA USAR EL TRIGGER PARA SPAWNEAR ENEMIGOS SUELTOS Y NO WAVES: LOS SPAWNERS DEBEN TENER CHEQUEADO EL BOOL TRIGGER SPAWN Y EL BOOL DEL ENEMY SPAWNER TRIGGER DESACTIVADO, 
EN EL TRIGGER QUE TIRA EL EVENTO (ENEMYSPAWNERTRIGGER) TAMBIEN HAY UN GAMEOBJECT EL CUAL ES EL CONTAINER DE TODOS LOS ENEMY SPAWNER TRIGGER QUE SE DESEEN USAR CON ESE TRIGGER

SI SE DEJAN ACTIVOS ENEMY SPAWN POINTS QUE NO SON UTILIZADOS, NO VA A TERMINAR LA SECCION, ESTAR ATENTO A ESO

EN EL SECTIONNODE HAY QUE PONER LOS TRIGGERS(ENEMYSPAWNERTRIGGER) QUE SE DESEEN USAR DENTRO DEL MISMO, SI NO SE PONEN NO SE REINICIARAN AL MORIR

SE PUEDEN HACER LA CANTIDAD DE TRIGGERS QUE SE QUIERAN CON LA CANTIDAD DE ENEMIGOS QUE SE DESEE, LO QUE NO SE PUEDE HACER ES SPAWNEAR MAS DE 1 SET DE 3 WAVES, POR AHORA,
SI SE QUIERE CAMBIAR ESO HAY QUE HABLARLO

SI SE QUIERE QUE SOLO SEA UNA WAVE LA SECTION Y SE PASA HAY QUE DEJAR DESACTIVADO EL START AT TRIGGER DEL SECTION NODE(SI SE LE AGREGA UN TRIGGER SUELTO, NO TERMINARA HASTA QUE TAMBIEN MATE ESOS ENEMIGOS)
